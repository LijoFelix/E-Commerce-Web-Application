# LUXE NOIR — Premium Dark E-Commerce Store

A fully functional dark-theme e-commerce platform built with React.

## Features
- Product catalog with 12 items, categories, search & sort
- Add to cart, cart drawer, full checkout flow
- User login & registration (Admin / User roles)
- Admin dashboard — product & order management
- Wishlist, order tracking, user profile
- Toast notifications, responsive design

## Getting Started

### Prerequisites
- Node.js 16+ installed ([nodejs.org](https://nodejs.org))

### Run Locally
```bash
# Install dependencies
npm install

# Start the development server
npm start
```

The app opens at **http://localhost:3000**

### Build for Production
```bash
npm run build
```
